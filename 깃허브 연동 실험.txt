방법 확인.

최대한 클린한 상태에서(이전에 git관련 파일이 있다면 지우고)

파워셸 터미널에 
git clone https://github.com/Temmie-Tem/CBB_web.git
git config --global user.name "이름"
git config --global user.email "이메일"

세 가지 입력.

이후 소스관리 탭을 확인.

정상적으로 연동되지 않을 경우, 확장 기능으로 가 "@builtin git"을 검색.
뜨는 확장기능들을 전부 활성화.

다시 처음부터 실행해 확인.

간혹 연동이 되어도 카테고리 정리가 엉키는 경우가 있음.

그건 알아서.

작업 후 파일을 저장하면 소스 관리 에서 변경 사항이 표시됨.
이때 변경한 내용을 몇 글자 정도로 요약 할 만한 메세지를 입력하고 코밋.
